# Campus Pulse Dashboard

##  Project Overview
Campus Pulse Dashboard is an interactive data analytics application that visualizes student satisfaction across campus facilities.

##  Features
- Data visualization (charts & graphs)
- Filtering options
- Insights and trends analysis
- Dashboard interface

##  Technologies Used
- Python
- Pandas
- Matplotlib / Visualization tools
- HTML, CSS (Frontend)

##  Project Structure
- backend → Data processing
- frontend → UI design
- data → Dataset
- notebook → Analysis

##  How to Run
1. Open project folder
2. Run main Python file / notebook
3. View dashboard output

##  Purpose
To analyze student satisfaction and help improve campus facilities using data insights.
